Git and Github

Git is an open source distributed version control system that helps developer manage projects with high speed and efficency. It allows you to moitor and work together with your team members at the same workspace. Github is to Git as what facebook is to the actual face. Github is designed as a Git repository hosting services like a storage, an online storage that allows you to keep track of and share Git version control projects outside of your local computer/server.

Importance of Git

  • Branching capabilities
  • Ability to track source code history
  • It serves as backup of your source code
  • Collaboration with other team members
  • Efficiency and can handle project of any size

HTML

HTML - HyperText Markup Language also known as the language of the web. It is not a programming language because programming languages deals with logic e.g. Conditionals(where you can say "if"). It's used to define the structure and content of a webpage. It comprises of tags and elements. The Word tags and elements are used interchangeably, the HTML tags are enclosed in angle brackets e.g . In HTML, we have inline and block elements. Inline elements are elements that takes up the width of the content, they do not accept margin or padding and cannot start on a new line e.g span, input, label, img . Block Elements, on the other hand are opposite of inline elements e.g paragraph, headings(h1, h2, h3, h4, h5, h6), div, ul , ol

Tags: Are entites used to wrap the content of a html element e.g headings, p, title, head etc. HTML attributes, gives addiional information about an element and are usually written in key-value pairs but not all attr are in key-value pairs e.g disable, autofocus and some accept multiple values e.g class. Comments, are used as references by developer to explain code content are usually ignored by the browser.

CSS

CSS - Cascading Style Sheets. It's basically use to add style and layout to an HTML document also it tells the browser how to present the document to the user. The importance of CSS to an HTML document is that, it makes the front-end of a website presentable and it creates user experience. Without CSS, websites would be less pleasing to the eye an likely much harder to navigate. There are 3 different types of styling in CSS.

Types of styling in CSS

There are three (3) major types of styling in CSS

Inline Styling
A method not recommended because of specificity. It basically adding style to an HTML element. It overrides any other type of styling in CSS.
Internal Styling
This method adds the style in the head element but because the browser loads fro top to bottom, this also is not a good recommendation especially for large project.
External Styling
This is written in a separate file .css and the is added in the element. Most recommended type of styling.

Specificity

Specificity is the means by which the browsers decides which CSS property values are most relevant to an element. Cascading means the order in which CSS rules are applied to an element.

Types of selectors:
Universal Selector: This selector selects all the elements on a webpage.
Element/type Selector: This selectors are less specific. It styles elements that have a certain type on that page
Class selector: Styles element that are specific to a particular class attribute value. A period (.) is used to specify a class.
ID Selector: Is used to style unique element. A hash symbol (#) is used to specify an id selector.
Attribute Selector: This selector styles content according to the attribute and the attribute value name mentioned enclosed in square brackets.

Content box

Content: The size of the area where the content is displayed is controlled through the width and height properties.

Padding:This is the outside of the content. It adds a white space around the content and it is controlled through the padding (top, right, bottom and left).

Border: This wraps the content and the padding. Its size is controlled through the border properties(border-width, border-style, border-color and border-radius

Margin: This wraps everything (the content, padding and border). It adds a white space around the elements and it size can be controlled through the margin properties (top, right, bottom and left)
